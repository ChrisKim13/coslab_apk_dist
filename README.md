  📱 COSLAB 테스터 설치 안내

  아래 링크에서 앱(APK)을 받아 설치해 주세요. (안드로이드 전용)

  1. 링크 열기 → APK 다운로드
  2. "출처를 알 수 없는 앱 설치" 허용 (휴대폰이 물어보면 허용)
  3. 설치 → 실행

  🔗 다운로드 (항상 최신 버전):
  https://github.com/ChrisKim13/coslab_apk_dist/releases/latest/download/COSLAB-
  tester.apk

  ⚠️  이전에 다른 버전이 깔려 있어 설치가 안 되면, 기존 COSLAB 앱을 삭제 후 다시 
  설치해 주세요.
